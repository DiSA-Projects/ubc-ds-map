spaCy is an open-source for advanced [[Natural Language Processing]] (NLP) designed  specifically for industrial-strength applications in Python and written in both Python and Cython for performance.

spaCy offers tokenization, Part-of-Speech (POS) Tagging, Dependency Parsing, Named Entity Recognition (NER), Lemmatization, Sentence Detection, and Integration with Deep Learning.
 
spaCy is built on Thinc and supports deep learning workflows, allowing connections with popular machine learning libraries like TensorFlow and PyTorch for advanced training and custom models
