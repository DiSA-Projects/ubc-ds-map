
# LANGUAGES

## HTML

HTML (Hypertext Markup Language) is used a web markup language to create websites and webpages. 
### General Tutorials

* [HTML Tutorial @ W3Schools](https://www.w3schools.com/html/) - Recommended, accessible, try-it-yourself sandbox
* [Structured HTML @ MDN Web Docs (Mozilla)](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Structuring_content) - Gold Standard, thorough, professional level
* [HTML Crash Course @ freeCodeCamp](https://www.freecodecamp.org/news/html-crash-course/) -Easy to follow, friendly tone, can apply for certificate

## CSS

CSS (Cascading Style Sheets) are used to "style" or apply design to webpage. If HTML defines the content organization on the page, CSS defines the look or appearance of that content. 

### General Tutorials

* [CSS Tutorial @ W3Schools](https://www.w3schools.com/css/) - Recommended, accessible, try-it-yourself sandbox
* [CSS Tutorial @ MDN Web Docs (Mozilla)](https://developer.mozilla.org/en-US/docs/Web/CSS/Tutorials) - Excellent, thorough
* [CSS Tutorial @ TutorialsPoint](https://www.tutorialspoint.com/css/index.htm) - Accessible, clear, easy-to-read

### Templates and Examples

* [CSS Zen Garden](https://csszengarden.com/) - Examples of different advanced uses of CSS. Templates can be downloaded and adapted.
## JavaScript

JavaScript is used not only for extending basic website functionality and interaction, but also to create more complicated web applications.

### General Tutorials

* [JavaScript Tutorial @ W3Schools](https://www.w3schools.com/Js/) - Recommended, accessible, try-it-yourself sandbox
* [The Modern JavaScript Tutorial (javascript.info)](https://javascript.info/) - Very comprehensive resource
* [JavaScript Tutorial @ MDN Web Docs (Mozilla)](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide) - Professional, comprehensive
